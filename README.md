# INVE-FAT SYSTEM

Sistema ERP empresarial moderno para gestión de inventario, logística, ventas, compras, contabilidad, recursos humanos, reportes, documentos, producción, ecommerce e inteligencia artificial.

## Objetivo

Crear una plataforma empresarial completa con diseño moderno, cálido y elegante, inspirada en sistemas ERP profesionales.

## Módulos principales

1. Dashboard Ejecutivo
2. Inventario y Almacén
3. Ventas
4. Compras
5. Contabilidad
6. Recursos Humanos
7. CRM
8. Producción
9. Proyectos
10. Documentos
11. Reportes y BI
12. Administración
13. Ecommerce
14. Comunicación
15. Inteligencia Artificial

## Tecnologías planificadas

- Frontend: React + Vite
- Backend: Node.js + Express
- Escritorio: Electron
- Base de datos: PostgreSQL
- Control de versiones: Git + GitHub
